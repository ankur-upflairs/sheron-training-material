🎨 CSS Basic Designing – Practical Assignment

📝 Objective:
Apply basic CSS properties to style HTML elements using:
- color
- background-color
- border
- height
- width

📂 Instructions:
1. Create an HTML file and link a CSS file.
2. Complete the following tasks using only HTML and CSS.
3. Use internal or external CSS (not inline styles).

✅ Tasks:

1️⃣ Create a `<div>` box with:
   - Width: 300px
   - Height: 150px
   - Background-color: lightblue
   - Border: 2px solid navy

2️⃣ Create a `<p>` paragraph inside the div with:
   - Text color: darkblue
   - Font-size: 16px

3️⃣ Create 3 buttons with:
   - Each having different background-color (e.g., red, green, yellow)
   - Width: 100px
   - Height: 40px
   - Border: none

4️⃣ Create a circle using a div:
   - Width: 100px
   - Height: 100px
   - Border-radius: 50%
   - Background-color: orange
   - Border: 2px dashed brown

5️⃣ Create a colored card layout:
   - Card width: 250px
   - Card height: 300px
   - Background: gradient (e.g., from blue to purple)
   - Border: 1px solid gray


📤 Deliverables:
- Submit your HTML & CSS files
- Ensure each task is commented in your code for identification
